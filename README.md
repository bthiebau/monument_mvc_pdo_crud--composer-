# MVC Monument avec composer

## Configuration

### Base de données

Créer un fichier `config/db.ini` avec les paramètres de configuration :

```ini
DB_DRIVER=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_NAME=mainbddpdo
DB_CHARSET=utf8mb4
DB_USER=root
DB_PASSWORD=""
```
> [!NOTE]
> Vous pouvez utiliser le fichier `config/db.template` comme modèle
